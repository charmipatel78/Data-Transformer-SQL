# 📊 Data Transformer - SQL Project

## 📌 Overview

Data Transformer is a MySQL project focused on data manipulation and analysis using advanced SQL concepts. It includes operations on customers, orders, and employees data to generate meaningful business insights.

---

## 🗂️ Database Structure

This project consists of the following tables:

* **Customers** – Stores customer details
* **Orders** – Contains order transactions
* **Employees** – Holds employee information

Database and queries are available in:

* `Data Transformer.sql` 

---

## ⚙️ Features

* Table creation with relationships (Foreign Keys)
* Joins (INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL JOIN using UNION)
* Subqueries and filtering
* Aggregate functions and conditions
* String functions (CONCAT, UPPER, LOWER, REPLACE, TRIM)
* Date functions (YEAR, MONTH, DATEDIFF, DATE_FORMAT)
* Window functions (RANK, RUNNING TOTAL)
* CASE statements for conditional logic

---

## 📈 Key SQL Operations

* Customer and order data merging using joins
* Finding customers with above-average orders
* Employee salary analysis
* Order date transformation and formatting
* Running total and ranking analysis
* Discount calculation using CASE

---

## 🛠️ Technologies Used

* MySQL
* SQL (DDL, DML, Advanced Queries)

---

## 🚀 How to Run

1. Open MySQL Workbench or Command Line
2. Run the SQL file:

   ```sql
   source Data Transformer.sql;
   ```
3. Execute queries step-by-step

## 📌 Conclusion

This project showcases advanced SQL skills including joins, analytical queries, and data transformation techniques used in real-world data analysis.
